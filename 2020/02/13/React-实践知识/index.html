TypingCard 组件的编写

TypingCard 是一个动态打字效果的组件

  • 比如单纯的文字打字效果

如果只是单纯的打印文字上面的函数就够了，但是想打印带标签的就需要再深入一下了，可以参考这篇博客

原理

  • 获取打印的内容
  • 将内容转化为字符和对象组成的数组中（对象是用来保存dom节点的）
  • 获取数组的第一个元素并在原数组中删除，然后判断：是字符则打印；是对象，则创建dom节点，并重复第一步
  • 数组长度为空时，判断对象是否有parent属性，有则回到第一步，没有则打印完毕
